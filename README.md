# ATO

## INSTALLATION
```
git clone --remote-submodules --recurse-submodules https://github.com/AutonomousTradingOrganization/frontend-react
cd frontend-react
```

### Install dependencies
```
bun install
```

### Run frontend
```
bun run start
```


#### Update programs (submodules)
```
git submodule update --remote
```
